<br>

> Full **Encoder-Decoder Transformer** reimplemented from "Attention Is All You Need" using only native PyTorch matrix operations — zero `nn.Transformer`, zero shortcuts. Every component hand-coded from the math.

<br>

| Metric | Result |
|:-------|:-------|
| 🚀 CUDA Kernel Speedup (N=64) | **2.71× vs CPU baseline** |
| ⏱️ Mean Inference Latency | **80.16 ms** |
| 📝 Inference Throughput | **382.41 chars/sec** |
| 🧪 Test Coverage | **9 suites · 100% SAT · 0 regressions** |
| 🏗️ Delivery | **13 sprints · MLflow tracked** |

<br>

**What was engineered from scratch:**
- ⚙️ Scaled Dot-Product Attention · Multi-Head Attention · Positional Encoding
- 🔧 Layer Normalization · Residual Connections · Position-wise Feed-Forward
- ⚡ Custom CUDA kernel (`attention_kernel.cu`) — mitigates global memory bottleneck
- 🌐 FastAPI inference server with input validation + health checks
- 📦 Full Docker deployment + MLflow experiment tracking
